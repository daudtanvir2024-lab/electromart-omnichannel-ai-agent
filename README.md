# ElectroMart AI Agent

## Overview

ElectroMart AI Agent is a production-ready OmniChannel AI Assistant built using n8n.

The system supports:

- AI Chatbot
- AI Voice Agent
- AI Calling Agent
- Product Search
- Product Recommendations
- Product Comparison
- Order Tracking
- CRM Automation
- Notifications
- Shopify Integration

## Tech Stack

- n8n
- Google Gemini
- OpenAI Whisper
- ElevenLabs
- Shopify
- Google Sheets
- Telegram
- Railway
