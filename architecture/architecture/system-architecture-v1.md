# ElectroMart AI Architecture V1

Customer communicates through:

- Website Chat
- Voice Message
- Phone Call

The request is processed by:

- Speech-to-Text (for voice)
- n8n
- Gemini/OpenAI

The AI Agent accesses:

- Shopify Product Data
- FAQ Knowledge Base
- CRM Data

The AI responds through:

- Chat
- Voice
- Phone Call

Notifications are sent through:

- Telegram
- Email
