# Product Database

## Database Type

Google Sheets

## Sheet Name

ElectroMart Products

## Worksheet

Products

## Columns

* Product_ID
* Product_Name
* Category
* Brand
* Price
* Stock
* Warranty
* Description
* Product_URL

## Purpose

This sheet will act as the temporary product database for the AI Agent.

In future versions, Google Sheets will be replaced with Shopify product data.
